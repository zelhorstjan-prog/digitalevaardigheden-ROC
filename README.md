# Digitale Vaardigheden

Een eenvoudige website met 9 modules om digitale vaardigheden te leren.

## Modules
1. Computer
2. Internet
3. E-mail
4. Veiligheid
5. Overheid & DigiD
6. Smartphone
7. Outlook
8. Microsoft Teams
9. OneDrive

## Online zetten via GitHub Pages
1. Upload alle bestanden naar een GitHub repository
2. Ga naar Settings → Pages
3. Kies bij Branch: **main** en klik Save
4. Uw website is live op: `https://uwgebruikersnaam.github.io/digitale-vaardigheden`
