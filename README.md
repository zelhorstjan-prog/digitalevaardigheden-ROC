# Digitale Vaardigheden

Een complete website met 9 interactieve modules om digitale vaardigheden te leren.
Alle modules zijn speciaal gemaakt voor NT2-taligen (Nederlands als tweede taal), niveau A1: korte zinnen, grote knoppen, plaatjes en directe feedback.

## Bestanden
- `index.html` — hoofdpagina met overzicht van alle 9 modules
- `computer.html` — Module 1: Computer (6 oefeningen)
- `internet.html` — Module 2: Internet (6 oefeningen)
- `email.html` — Module 3: E-mail (6 oefeningen)
- `veiligheid.html` — Module 4: Veiligheid (6 oefeningen)
- `overheid.html` — Module 5: Overheid & DigiD (6 oefeningen)
- `smartphone.html` — Module 6: Smartphone (6 oefeningen)
- `outlook.html` — Module 7: Outlook (6 oefeningen)
- `teams.html` — Module 8: Microsoft Teams (6 oefeningen)
- `onedrive.html` — Module 9: OneDrive (6 oefeningen)
- `README.md` — deze beschrijving

## Alle modules zijn compleet ✅
1. Computer
2. Internet
3. E-mail
4. Veiligheid
5. Overheid & DigiD
6. Smartphone
7. Outlook
8. Microsoft Teams
9. OneDrive

Elke module heeft:
- Een introductiescherm
- 6 interactieve, aanklikbare oefeningen
- Directe feedback (groen ✅ / rood ❌)
- Een voortgangsbalk
- Een eindscherm met sterren-score
- Een knop om de les opnieuw te doen

## Online zetten via GitHub Pages
1. Upload **alle bestanden** naar een GitHub repository
2. Ga naar Settings → Pages
3. Kies bij Branch: **main** en klik Save
4. Uw website is live op: `https://uwgebruikersnaam.github.io/digitale-vaardigheden`
